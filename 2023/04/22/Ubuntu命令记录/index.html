Ubuntu命令记录

2023/4/22 学习 Ubuntu

持续补充中…

ssh

连接远程服务器

ssh USER@ip

系统

搜索和过滤

# 过滤命令返回的结果
COMMAND | grep KEYWORD
# findstr是windows的命令，功能和grep差不多

网络

# 查看指定端口有没有开启
# 切换到root用户才可以看到所有信息，否则只能看到当前用户的信息
sudo -i
netstat -ap | grep PORT

进程

# 杀死进程
kill PID
# 或者
kill -9 PID

用户

# 添加用户
sudo useradd USER

# 删除用户
sudo deluser --remove-home USER

# 切换普通用户
sudu su - USER

# 切换根用户
sudu -i

# 查看用户列表
cat /etc/passwd

# 查看用户组列表
cat /etc/group

文件系统权限

# 修改目录所属用户
sudo chown -R USER[:GROUP] FOLDER

# 修改目录权限
sudo chmod -R 777 FOLDER

守护（后台）进程

# 重载守护进程配置
sudo systemctl daemon-reload

# 启动指定守护进程
sudo systemctl start DAEMON

# 查看指定守护进程状态
sudo systemctl status DAEMON

# 守护进程自启动
sudo systemctl enable DAEMON

分屏

# 创建新分屏
screen -S SCREENNAME

# 查看当前用户下的所有分屏
screen -ls

# 退出当前分屏，停止运行
exit

# 进入指定分屏
screen -r PROCESSID

退出当前分屏，继续运行
ctrl + a 同时 ctrl + d

编译

# 需要当前目录下有Makefile文件
make

常用命令行软件

vim

搜索关键字
/keyword

模糊搜索关键字,搜索到了之后按”n”键查看下一个匹配项，按”N”查看上一个匹配项
?keyword

不保存退出
:q!

nano

保存文件
ctrl + o

保存退出文件
ctrl + x
